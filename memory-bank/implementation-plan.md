# AI求职助手 - 实施计划

> 本计划基于 PRD V1.0 (MVP) 范围，共分为 15 个步骤。每步包含明确的任务描述和验证测试。

---

## Step 1: 项目初始化与基础设施

**任务：**
- 用 `create-next-app` 初始化 Next.js 14 项目（App Router, TypeScript, Tailwind CSS, ESLint）
- 安装核心依赖：shadcn/ui, Prisma, NextAuth, Zustand, Zod
- 配置 Docker Compose（PostgreSQL + Redis）
- 配置 Prisma，连接 PostgreSQL
- 创建基础布局组件（导航栏、侧边栏、主内容区）
- 配置 ESLint + Prettier

**测试：**
- [ ] `npm run dev` 启动成功，首页可访问
- [ ] `docker compose up` 启动 PostgreSQL 和 Redis
- [ ] Prisma 可以连接数据库（`npx prisma db push` 成功）
- [ ] 基础布局在桌面端和移动端正常显示

---

- 创建登录/注册页面
- 实现受保护路由中间件
- 创建用户 Dashboard 首页（空壳页面，展示导航入口）

**测试：**
- [ ] `npx prisma migrate dev` 成功创建所有表
- [ ] 可以注册新用户并登录
- [ ] 未登录用户访问 Dashboard 被重定向到登录页
- [ ] 登录后能看到 Dashboard 首页，左侧导航可以点击

## Step 3: 简历上传与文件管理

**任务：**
- 实现文件上传组件（拖拽 + 点击上传）
- 支持 PDF 和 DOCX 格式，限制 10MB

**测试：**

- [ ] 上传非 PDF/DOCX 格式时提示错误
- [ ] 上传成功后数据库中有对应记录
- [ ] 简历列表页面能展示已上传的简历

## Step 4: AI 简历解析

**任务：**
- 集成 pdf-parse 库提取 PDF 文本
- 集成 mammoth 库提取 DOCX 文本
- 调用 Claude API，将提取的文本解析为结构化 JSON（按 PRD 定义的简历数据结构）
- 创建简历解析结果展示页面（结构化展示：基本信息、教育背景、经历、技能、获奖）
- 用户可以手动修正解析结果（每个字段可编辑）
- 解析结果保存到数据库

**测试：**
- [ ] 上传一份真实简历 PDF，能在 30 秒内完成解析
- [ ] 解析结果包含完整的结构化字段（姓名、学校、经历等）
- [ ] 可以修改解析结果中的字段并保存
- [ ] 保存后重新打开页面，数据正确回显

## Step 8: 智能岗位匹配

**任务：**
- 集成 OpenAI Embeddings API
- 为简历内容和岗位 JD 生成 embedding 向量
- 安装并配置 pgvector 扩展
- 实现匹配算法：
  - 技能匹配（30%）：关键词重合度
  - 经历相关性（25%）：embedding 余弦相似度
  - 硬性条件（20%）：学历/城市/类型规则匹配
  - 行业/方向（15%）：意向匹配
  - 优先条件（10%）：加分项匹配
- 岗位列表页增加匹配度百分比显示
- 按匹配度降序排列
- 每个岗位卡片展示匹配/不匹配项说明

**测试：**
- [ ] 设置求职意向后，60 秒内返回匹配结果
- [ ] 匹配度百分比合理（技能高度相关的岗位分数高）
- [ ] 匹配详情能展示具体的匹配/不匹配维度
- [ ] 修改求职意向后，匹配结果相应变化

## Step 9: AI 简历优化

**任务：**
- 用户在岗位详情页点击「针对此岗位优化简历」
- 调用 Claude API，输入当前简历 + 目标岗位 JD，生成优化方案（按 PRD 定义的修改方案结构）
- 优化方案展示页面（分区展示：整体策略、逐条修改、关键词补充、经历取舍、排版建议）
- 用户可以逐条接受/拒绝修改建议（checkbox）
- 「一键应用」按钮，根据接受的建议生成新版本简历
- 修改前后 diff 对比视图

**测试：**
- [ ] 点击优化后 45 秒内生成方案
- [ ] 方案包含完整的五个部分（整体策略、逐条修改、关键词补充、经历取舍、排版建议）
- [ ] 可以单独接受/拒绝每条建议
- [ ] 应用建议后生成新版本简历，版本列表中可见
- [ ] diff 视图正确展示修改前后的差异

**测试：**

## Step 13: 模拟面试 - 三轮面试与评估

**任务：**
- 实现三轮面试的差异化 Prompt：
  - 第一轮（技术/业务基础面）：60% 简历深挖 + 40% 八股基础
  - 第二轮（深度思维面）：50% 追问决策逻辑 + 30% 开放性问题 + 20% 行业认知
  - 第三轮（HR综合面）：稳定性 + 价值观 + 综合素养 + 压力测试
- 实现五种面试官风格的差异化 Prompt
- 面试结束后生成评估报告：
  - 各题得分（1-10）
  - 回答亮点
  - 改进建议
  - 参考答案/答题框架
  - 综合评级
- 评估报告页面展示
- 面试历史记录列表

**测试：**
- [ ] 三轮面试的问题风格明显不同
- [ ] 五种面试官风格体验有差异（温和型会给提示，压力型会 challenge）
- [ ] 面试结束后生成评估报告，各题有评分和建议
- [ ] 面试历史记录列表能查看之前的面试
